# Arkanna أركاننا

Arkanna is a modern ecommerce platform designed for a Moroccan women's cooperative that produces natural argan-based cosmetic products.

The platform highlights Moroccan craftsmanship, natural beauty products, and the women behind traditional argan oil production.

The goal of the project is to create a premium, culturally authentic digital experience for organic beauty products.

---

## 🌿 Features

- Modern ecommerce interface
- Bilingual support (French / Arabic)
- RTL layout for Arabic
- Product catalog and product details
- Shopping cart UI
- Testimonials
- Newsletter subscription
- Interactive product sliders
- Smooth animations with Framer Motion
- Responsive mobile design

---

## 🛍 Product Categories

- Argan Oil
- Skin Care
- Hair Care
- Natural Soaps
- Gift Boxes
